# ⚡ Smart Electrical Theft Detection System

## 📌 Overview
This is a web-based system that detects electricity theft using simulated data without hardware.

## 🎯 Features
- Real-time electricity monitoring
- Automatic theft detection
- Alert popup system
- Web-based UI dashboard

## 🛠️ Technologies Used
- Python
- Flask
- HTML, CSS, JavaScript

## 📂 Project Structure
project/
│── app.py
│── simulator.py
│── detection.py
│── templates/
│── static/
│── data/

## 🚀 How to Run

### Step 1: Install Flask
pip install flask

### Step 2: Run Simulator
python simulator.py

### Step 3: Run Web App
python app.py

### Step 4: Open Browser
http://127.0.0.1:5000/

## 🧠 Working
The system simulates electricity usage and detects abnormal spikes to identify theft.
